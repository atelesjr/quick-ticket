
## Quick-ticket

This is simple application to create and list tickets set with log events - Sentry. 
 
#Stack: 
Next.js, React.js 
database: Neon, Prisma 
CSS: Tailwind 
Database: Prisma ORM 
Monitory: Sentry 
Authentication: JWT 


```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
