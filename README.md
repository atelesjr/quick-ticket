
## Quick-ticket

This is simple application to create and list tickets set with log events - Sentry. 
 
#Stack: 
- Next.js, React.js 
- database: Neon, Prisma ORM
- CSS: Tailwind  
- Monitory: Sentry 
- Authentication: JWT 


```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
